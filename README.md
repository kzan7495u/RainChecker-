# 🌧️ Rain Checker

A Node.js app that checks whether it will rain tomorrow in a selected location using real-time weather API data.

## 🚀 Features

- Check if it will rain tomorrow in any city
- Location-based weather forecast
- Uses real-time weather API data
- Simple and fast response (rain / no rain result)

## 🛠️ Tech Stack

- Node.js
- Express.js
- Axios / Fetch API
- dotenv

## ⚙️ Installation

- Clone the repo
- git clone <your-repo-link>
- npm install
- API_KEY=your_api_key_here
- node app.js

The app fetches weather forecast data for the selected location and analyzes tomorrow’s weather condition to determine whether rain is expected or not.
